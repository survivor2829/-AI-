# 新增产品模板 SOP

## 需要准备的文件

- `screen2.jpg`：第2屏固定图（产品卖点展示图，通常是场景/功能特写）
- `screen4.jpg`：第4屏固定图（清洁对比图，通常是使用效果展示）
- 建议尺寸：750px 宽，图片来自公司现有详情页截图

## 操作步骤

1. 在 `templates/` 下新建文件夹，名称就是产品类型，例如：
   ```
   templates/高压清洗机/
   ```

2. 把准备好的两张图放进去，文件名必须完全一致：
   ```
   templates/高压清洗机/screen2.jpg
   templates/高压清洗机/screen4.jpg
   ```

3. 重启服务：
   ```
   python app.py
   ```

4. 刷新浏览器，第3步"选择模板"页面会自动出现新模板卡片，无需改代码。

5. 选择该模板，点"下一步"生成图片，第2屏和第4屏会自动使用新放入的图片。

## 注意事项

- **文件夹名称必须和前端第2步填写的"产品类型"字段一致**，否则模板找不到
  - 例如：模板目录叫 `高压清洗机`，前端产品类型也要填 `高压清洗机`
  - 目前已有模板：`扫地车`、`洗地车`
- 只需要 `screen2.jpg` 和 `screen4.jpg` 两个文件，其他文件不影响功能
- 不需要修改任何代码，放文件重启即可
- 图片格式支持 `.jpg`、`.jpeg`、`.png`，优先级按此顺序查找

## 验证是否生效

生成后查看终端日志，出现以下内容说明模板生效：

```
GET /api/template-thumb/高压清洗机/screen2.jpg 200
GET /api/template-thumb/高压清洗机/screen4.jpg 200
```

若出现 `[警告] 未找到 templates/xxx/screen2.jpg`，说明目录名或文件名拼写有误。
